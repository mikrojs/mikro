# @mikrojs/waveshare

Waveshare board definitions for Mikro.js.

## Supported boards

### ESP32-S3-Knob-Touch-LCD-1.8

1.8" round touch display with rotary encoder, powered by ESP32-S3.

```ts
import {board} from '@mikrojs/waveshare/esp32-s3-knob-touch-lcd-1.8'
```

Uses the [`@mikrojs/driver-sh8601`](../driver-sh8601) QSPI display driver (native, compiled into firmware).
