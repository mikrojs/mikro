# Waveshare Display

Animated rainbow color bars on the Waveshare ESP32-S3-Knob-Touch-LCD-1.8" round display.

```sh
npm create mikrojs my-waveshare --template waveshare-display
```

## Hardware

- [Waveshare ESP32-S3-Knob-Touch-LCD-1.8](https://www.waveshare.com/wiki/ESP32-S3-Knob-Touch-LCD-1.8) (1.8" round touch display with rotary encoder)
- USB-C cable

## Run

```sh
npx mikro dev       # develop on connected device
npx mikro deploy    # build and deploy to device
```
